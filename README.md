# corona_game_server
## To use the DB, please create a .env file like: 
```
JWTsecret=<your secret>
mongoURI=<your Mongo URI>
```
## Run the project locally by
```
./build.sh
```
Then open `localhost:5000/login` and sign up an admin account.
